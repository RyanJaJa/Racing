開發環境
====

* JDK 1.8
    - 請先確定環境變數JAVA_HOME設定正確
* Intellij Idea
https://www.jetbrains.com/idea/

* mariadb (mysql)

## 開啟專案
---
- 打開Intellij -> Open -> 選擇專案資料夾
- 出現Import Project from Gradle視窗
    - 勾選 Use auto-import
    - ok
    - 初次開啟專案，會下載所使用的libraries，請耐心等候       
    
## 設定SDK
---
- File -> Project Structure 
    - SDKS -> + -> JDK -> 選擇JDK路徑
    - Project -> ProjectSDK 選剛剛設定的1.8, Language Level 選8 
    - OK
